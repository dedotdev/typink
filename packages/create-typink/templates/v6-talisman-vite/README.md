# Typink Dapp Template

### Getting started

To get started, install dependencies & start the application

```shell
# With yarn
yarn install && yarn start

# With npm
npm install && npm run start

# With pnpm
pnpm install && pnpm run start

# With bun
bun install && bun start
```
